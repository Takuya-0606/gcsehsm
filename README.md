# gcsehsm
developede when master course student in 2026

## インストール

Node.js (18 以上推奨) を用意した上で、依存関係をインストールしてください。

```bash
npm install
```

## 実行

ローカル環境でアプリケーションを起動します。

```bash
npm run dev
```

## 例

以下の入力ファイルを用意して読み込ませると、熱力学プロパティの計算結果を確認できます。

- `tests/freqsection/scale_1.2.out`
- `tests/volumesection/alpha_1.0/volume.out`
- `tests/volumesection/alpha_1.2/volume.out`
- `tests/dipolesection/gas/pol_gas.out`
- `tests/dipolesection/liquid/pol_aq_scale_1.2.out`
